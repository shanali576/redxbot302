## 🚀 Quick Setup

### ① Get Your Session ID

<div align="center">
  <a href="https://redxpair3.vercel.app" target="_blank">
    <img src="https://img.shields.io/badge/🔑_GET_SESSION_ID_NOW-FF0000?style=for-the-badge&logoColor=white" alt="Get Session ID"/>
  </a>
</div>

<br/>

Visit **[redxpair3.vercel.app](https://redxpair3.vercel.app)** → Enter your number → Get your `SESSION_ID`

---

### ② Clone & Configure

```bash
git clone https://github.com/AbdulRehman19721986/redxbot302.git
cd redxbot302
cp sample.env .env
nano .env
# Paste your SESSION_ID and configure your number
```

---

### ③ Install & Run

```bash
npm install
node --expose-gc index.js
```

---

### ④ Rebrand (Optional — run once)

```bash
node rebrand.js
```

## ⚙️ Environment Variables

| Variable | Required | Description |
|---|---|---|
| `SESSION_ID` | ✅ **Required** | From redxpair3.vercel.app |
| `OWNER_NUMBER` | ✅ **Required** | Your WhatsApp number (no `+`) |
| `BOT_NAME` | Optional | Display name (default: `REDXBOT302`) |
| `PREFIX` | Optional | Command prefix (default: `.`) |
| `COMMAND_MODE` | Optional | `public` or `private` |
| `TIMEZONE` | Optional | Default: `Asia/Karachi` |
| `APP_URL` | Optional | For Heroku/Render keep-alive |
| `PAIRING_NUMBER` | Optional | If not using session file |
